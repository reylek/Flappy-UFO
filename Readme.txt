В проекте использовалась библиотека SFML (https://www.sfml-dev.org/).
Для сборки проекта необходимо выбрать конфигурацию проекта "Release", платформу "x86" и настроить проект для SFML (https://www.sfml-dev.org/tutorials/2.5/start-vc.php).

Выполняемый .exe файл Google drive не дает скачать как вредоностное ПО, его можно скачать по ссылке на GitHub: https://github.com/reylek/Flappy-UFO
